# Coins

### ZORA's Coins Protocol

ZORA's Coins Protocol is a protocol for creating and managing ERC-20 media coins.

These coins support:

    * Multiple owners
    * Customizable payout addresses
    * Customizable media via EIP-7572
    * Fair markets with Uniswap V3 and locked liquidity
    * Creator earnings from primary and secondary trading

These contracts along with our coins SDK allows anyone to deploy a ZORA coin in seconds.
